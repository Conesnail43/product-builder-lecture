About

사이트 소개

Jerry's Test Sandbox는 재미있고 유용한 웹 도구를 실험하고 공유하기 위해 만든 개인 프로젝트입니다. 복잡한 가입이나 설치 없이, 브라우저에서 바로 사용할 수 있는 도구들을 제공합니다.

도구 소개

🎱

번호 추출기

1부터 45까지의 숫자를 Fisher–Yates 알고리즘으로 무작위 셔플하여 로또 번호를 생성합니다. 1게임 또는 5게임을 한번에 뽑을 수 있으며, 번호는 색상으로 구간을 구분합니다. 재미로 사용하는 번호 생성 도구이며, 당첨 확률에는 영향을 주지 않습니다.

🐾

동물상 판독기

Google Teachable Machine으로 학습된 이미지 분류 모델을 활용해 사진 속 인물의 동물상을 분석합니다. 강아지, 고양이, 사슴, 공룡 네 가지 라벨 중 가장 유사도가 높은 결과를 확률과 함께 보여줍니다. 카메라 촬영 또는 사진 업로드 두 가지 방식을 지원하며, 모든 분석은 브라우저 내에서 이루어집니다.

📜

역사적 사실

고대사부터 근현대까지, 잘 알려지지 않았거나 놀라운 역사적 사실을 랜덤으로 보여줍니다. 30개 이상의 사실이 수록되어 있으며, 중복 없이 순환하여 매번 새로운 이야기를 만날 수 있습니다. 역사에 대한 흥미와 호기심을 자극하는 것을 목표로 합니다.

만든 이유

웹 기술을 직접 실험하고, 완성된 결과물을 다른 사람들과 공유하고 싶어서 시작했습니다. 각 도구는 실용성보다는 재미와 흥미에 초점을 맞추었으며, 사용자가 부담 없이 즐길 수 있도록 설계했습니다.

기술 스택

  • HTML / CSS / JavaScript: 별도의 프레임워크 없이 순수 웹 기술로 구현
  • Google Teachable Machine: 동물상 분류 모델 학습 및 배포
  • TensorFlow.js: 브라우저 내 AI 모델 실행
  • Disqus: 댓글 기능
  • Formspree: 문의 폼 처리
  • Google AdSense: 광고 서비스
